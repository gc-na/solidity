# المُنشئ (Constructor) في لغة Solidity: التعريف والاستخدامات

## الملخص
المُنشئ (Constructor) في Solidity هو دالة خاصة تُستخدم لإنشاء حالة جديدة من العقد الذكي، وتعتبر الوسيلة الأساسية لتهيئة المتغيرات القابلة للتعديل عند نشر العقد.

## الوثائق
المُنشئ هو دالة تُستدعى مرة واحدة فقط عند نشر العقد الذكي. الهدف الرئيسي من المُنشئ هو تهيئة حالة العقد وتحديد القيم الأولية للمتغيرات. يُمكن تعريف المُنشئ باستخدام الكلمة المحجوزة `constructor`، ويدعم المُنشئ معلمات يمكن تمريرها عند نشر العقد.

### الاستخدام
- يُكتب المُنشئ داخل جسم العقد.
- يمكن أن يحتوي المُنشئ على معلمات لتخصيص متغيرات الحالة.
- لا يُمكن استدعاء المُنشئ بعد نشر العقد، ويعمل بشكل تلقائي عند إنشائه.

### التفاصيل
- يُمكن أن يحتوي المُنشئ على منطق إضافي، مثل التحقق من الشروط قبل تهيئة المتغيرات.
- إذا لم يتم تعريف مُنشئ، يقوم المُترجم تلقائيًا بإنشاء مُنشئ فارغ للعقد.
- يمكن أن يكون المُنشئ غير مُعدل (غير قابل للتعديل) أو مُعدل (قابل للتعديل) حسب الحاجة.

## الأمثلة
### مثال أساسي لمُنشئ
```solidity
pragma solidity ^0.8.0;

contract SimpleStorage {
    uint256 public storedData;

    constructor(uint256 initialValue) {
        storedData = initialValue;
    }
}
```
في هذا المثال، يتم استخدام المُنشئ لتعيين قيمة أولية للمتغير `storedData` عند نشر العقد.

### مثال مع شروط
```solidity
pragma solidity ^0.8.0;

contract LimitedStorage {
    uint256 public storedData;
    address public owner;

    constructor(uint256 initialValue) {
        require(initialValue > 0, "Initial value must be greater than 0");
        storedData = initialValue;
        owner = msg.sender;
    }
}
```
في هذا المثال، يتم التحقق من أن القيمة الأولية أكبر من 0 قبل تعيينها.

## الشرح
### الأخطاء الشائعة
- **عدم تعيين القيم الأولية:** إذا نسيت تعيين القيم الأولية في المُنشئ، قد يؤدي ذلك إلى إظهار القيم الافتراضية، مما قد يؤثر على منطق العقد.
- **عدم استخدام الشروط:** عدم إضافة شروط في المُنشئ قد يؤدي إلى حالات غير متوقعة.

### ملاحظات إضافية
- يجب أن يكون المُنشئ خاليًا من أي عمليات مكلفة، حيث يتم تنفيذ المُنشئ مرة واحدة فقط.
- يُفضل الحفاظ على بساطة المنطق داخل المُنشئ لتجنب التعقيد.

## ملخص من جملة واحدة
المُنشئ في Solidity هو دالة خاصة تُستخدم لتهيئة حالة العقد الذكي عند نشره وتعيين القيم الأولية للمتغيرات.
