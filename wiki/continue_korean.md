# Solidity의 "continue" 명령어: 사용법 및 예제

## 개요
Solidity에서 "continue" 명령어는 반복문 내에서 현재 반복의 나머지 부분을 건너뛰고 다음 반복으로 이동하기 위해 사용됩니다. 이 기능은 특정 조건에서 반복의 흐름을 제어하는 데 유용합니다.

## 문서
"continue" 명령어의 주요 목적은 반복문에서 조건이 충족되지 않을 때 나머지 코드를 실행하지 않고 다음 반복으로 넘어가는 것입니다. 주로 `for`, `while`, `do while`과 같은 반복문에서 사용됩니다. 

### 사용법
- **구문**: `continue;`
- **위치**: 반복문 내부의 조건문과 함께 사용됩니다.
- **기능**: 특정 조건이 참일 때, 현재 반복의 나머지 부분을 무시하고 다음 반복으로 진행합니다.

### 예시
```solidity
pragma solidity ^0.8.0;

contract Example {
    function testContinue() public pure returns (uint) {
        uint sum = 0;
        for (uint i = 0; i < 10; i++) {
            if (i % 2 == 0) {
                continue; // 짝수일 경우 현재 반복 건너뛰기
            }
            sum += i; // 홀수일 경우에만 합산
        }
        return sum; // 1 + 3 + 5 + 7 + 9 = 25
    }
}
```

## 설명
"continue" 명령어를 사용할 때 주의해야 할 점은, 이 명령어가 반복문의 흐름을 변경한다는 것입니다. "continue"를 사용할 경우, 조건이 충족되는 동안 해당 반복의 나머지 코드는 실행되지 않으므로, 의도한 대로 흐름을 제어할 수 있습니다.

### 일반적인 실수 및 주의사항
- "continue"를 적절한 조건과 함께 사용하지 않을 경우, 원하는 결과가 나오지 않을 수 있습니다. 예를 들어, 조건이 항상 참이면 무한 루프에 빠질 수 있습니다.
- Solidity의 gas 비용을 고려하여 불필요한 반복과 조건문을 피하는 것이 좋습니다.

## 한 줄 요약
Solidity에서 "continue" 명령어는 반복문 내에서 특정 조건을 만족할 때 현재 반복을 건너뛰고 다음 반복으로 이동하는 데 사용됩니다.
