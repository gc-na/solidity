# المعدلات (Modifiers) في لغة سوليديتي: دليل شامل

## الملخص
المعدلات (Modifiers) في لغة سوليديتي هي أدوات قوية تسمح للمطورين بتعديل سلوك الدوال، مما يسهل إدارة الوصول والتحكم في تنفيذ الكود.

## الوثائق
### الغرض من المعدلات
تستخدم المعدلات في سوليديتي للتحقق من الشروط قبل تنفيذ دالة معينة. يمكن استخدامها لتنفيذ عمليات مثل التحقق من هوية المستخدم أو التأكد من أن الحالة الحالية للعقد تسمح بتنفيذ الدالة.

### الاستخدام
لتعريف معدل، يتم استخدام الكلمة المفتاحية `modifier`، تليها اسم المعدل وتعريف الشروط المطلوبة. يمكن تطبيق المعدل على دالة باستخدام الكلمة المفتاحية `modifier` قبل تعريف الدالة. إذا تم استيفاء الشروط، يتم تنفيذ الكود داخل المعدل، ومن ثم يتم تنفيذ الكود داخل الدالة.

### التفاصيل
- يمكن استخدام المعدلات لتقليل تكرار الكود.
- يمكن أن تحتوي على شروط معقدة.
- يمكن استخدامها لتعديل الحالة قبل تنفيذ الدالة.
- يمكن استخدام عدة معدلات في دالة واحدة.

## الأمثلة
### مثال أساسي 1: معدل بسيط للتحقق من الملكية
```solidity
pragma solidity ^0.8.0;

contract MyContract {
    address public owner;

    constructor() {
        owner = msg.sender;
    }

    modifier onlyOwner() {
        require(msg.sender == owner, "ليس المالك!");
        _;
    }

    function restrictedFunction() public onlyOwner {
        // تنفيذ العمليات المسموح بها فقط للمالك
    }
}
```

### مثال أساسي 2: معدل للتحقق من حالة معينة
```solidity
pragma solidity ^0.8.0;

contract Voting {
    bool public votingOpen;

    modifier onlyWhileOpen() {
        require(votingOpen, "التصويت مغلق.");
        _;
    }

    function openVoting() public {
        votingOpen = true;
    }

    function vote() public onlyWhileOpen {
        // تنفيذ عملية التصويت
    }
}
```

## الشرح
### الأخطاء الشائعة
- **نسيان استخدام الكلمة `_`:** عند تعريف المعدل، يجب دائمًا استخدام `_` في المكان الذي تريد تنفيذ الكود الأصلي للدالة.
- **عدم التحقق من الشروط بشكل صحيح:** تأكد من أن الشروط في المعدل دقيقة لتجنب الأخطاء عند تنفيذ الدالة.
- **استخدام معدلات متعددة:** تأكد من ترتيب المعدلات بشكل صحيح إذا كنت تستخدم أكثر من معدل في دالة واحدة.

## ملخص في جملة واحدة
المعدلات في سوليديتي هي ميزة قوية للتحكم في سلوك الدوال والتحقق من الشروط قبل تنفيذها، مما يعزز الأمان والكفاءة في العقود الذكية.
