# الكلمة المفتاحية "external" في لغة Solidity: فهم دقيق واستخدام فعال

## ملخص
الكلمة المفتاحية "external" في Solidity تُستخدم لتعريف أنواع معينة من الدوال التي يمكن استدعاؤها من خارج العقد الذكي. تعتبر هذه الكلمة مفيدة لتحسين كفاءة استدعاء الدوال، حيث تختلف عن "public" في كيفية تنفيذها.

## الوثائق
تُستخدم الكلمة المفتاحية "external" لتحديد دوال يمكن استدعاؤها فقط من خارج العقد، مما يعني أنها لا يمكن استدعاؤها من داخل نفس العقد. تُعتبر هذه الكلمة مناسبة للدوال التي تحتاج إلى استدعاء من عقود أو حسابات أخرى، وتساعد في تقليل تكلفة الغاز عند استدعاء الدوال.

### الاستخدام
لإعلان دالة باستخدام "external"، يجب كتابة الكود بالشكل التالي:

```solidity
pragma solidity ^0.8.0;

contract MyContract {
    externalFunction() external {
        // تنفيذ الدالة
    }
}
```

### التفاصيل
- **الكفاءة**: عند استخدام "external"، يتم تمرير البيانات بشكل مباشر بدلاً من نسخها، مما يؤدي إلى تقليل تكلفة الغاز.
- **الاستخدامات**: تُستخدم بشكل شائع في العقود الذكية التي تتطلب تفاعلات مع عقود أو حسابات أخرى.
- **القيود**: لا يمكن استدعاء دالة "external" من داخل نفس العقد باستخدام اسم الدالة مباشرة. يجب استخدام `this` للإشارة إلى الدالة.

## الأمثلة
### مثال 1: دالة بسيطة
```solidity
pragma solidity ^0.8.0;

contract MyContract {
    uint public value;

    function setValue(uint _value) external {
        value = _value;
    }
}
```
في هذا المثال، يمكن استدعاء دالة `setValue` من خارج العقد لتعيين قيمة.

### مثال 2: استدعاء باستخدام `this`
```solidity
pragma solidity ^0.8.0;

contract MyContract {
    uint public value;

    function setValue(uint _value) external {
        value = _value;
    }

    function callSetValue(uint _value) public {
        // استدعاء الدالة باستخدام this
        this.setValue(_value);
    }
}
```
هنا، يتم استدعاء دالة `setValue` من داخل العقد باستخدام `this`.

## الشرح
### العوائق الشائعة
- **عدم القدرة على الاستدعاء الداخلي**: يجب تذكر أنه لا يمكن استدعاء الدوال "external" من داخل نفس العقد مباشرة. 
- **أداء الغاز**: قد تؤدي الدوال "external" إلى تقليل تكلفة الغاز عند التعامل مع كميات كبيرة من البيانات.
- **عدم الحاجة إلى معاملات**: لا يمكن للدوال "external" أن تُستدعى من كود Solidity مباشرة دون استخدام `this`.

## ملخص في جملة واحدة
الكلمة المفتاحية "external" في Solidity تُستخدم لتعريف دوال يمكن استدعاؤها فقط من خارج العقد، مما يحسن كفاءة استدعاء الدوال.
