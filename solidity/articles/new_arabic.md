# استخدام الكلمة المفتاحية "new" في لغة Solidity

## ملخص
تُستخدم الكلمة المفتاحية "new" في Solidity لإنشاء مثيلات جديدة من العقود الذكية أو الهياكل (Structs) في شبكة البلوكشين. تعتبر هذه الكلمة جزءًا أساسيًا من بناء العقود الذكية وتمكن المطورين من إدراج كود أكثر تعقيدًا وفعالية.

## الوثائق
### الغرض
الكلمة "new" تُستخدم لإنشاء كائن جديد من نوع معين، سواء كان ذلك عقدًا ذكيًا أو هيكلًا. هذا يتيح لك استدعاء العقود الذكية من داخل عقود أخرى، مما يعزز التفاعل بين العقود ويسمح بتطوير تطبيقات أكثر تعقيدًا.

### الاستخدام
يمكن استخدام "new" عند تعريف متغير من نوع عقد أو هيكل. على سبيل المثال، عند إنشاء عقد جديد، يمكنك استخدام "new" لتخصيص كائن من ذلك العقد.

### التفاصيل
عند استخدام "new" لإنشاء مثيل جديد من عقد، يتم تنفيذ مُنشئ العقد (Constructor)، مما يتيح لك تمرير المعاملات اللازمة أثناء إنشاء العقد. يجب أن يكون العقد الذي تقوم بإنشائه قابلًا للتنفيذ (deployable) ويجب أن يكون لديك الصلاحيات اللازمة لإنشاءه.

## أمثلة
### مثال 1: إنشاء مثيل جديد لعقد
```solidity
pragma solidity ^0.8.0;

contract SimpleStorage {
    uint256 public storedData;

    constructor(uint256 initialValue) {
        storedData = initialValue;
    }
}

contract Factory {
    SimpleStorage public simpleStorage;

    function createSimpleStorage(uint256 initialValue) public {
        simpleStorage = new SimpleStorage(initialValue);
    }
}
```
### مثال 2: إنشاء مثيل جديد من هيكل
```solidity
pragma solidity ^0.8.0;

contract StructExample {
    struct Person {
        string name;
        uint age;
    }

    Person public person;

    function createPerson(string memory _name, uint _age) public {
        person = Person(_name, _age);
    }
}
```

## الشرح
### الأخطاء الشائعة
- **عدم توافق المعاملات**: تأكد من تمرير المعاملات الصحيحة إلى مُنشئ العقد عند استخدام "new". عدم تطابق المعاملات قد يؤدي إلى فشل التنفيذ.
- **عدم وجود موارد كافية**: عند إنشاء عقود جديدة، تأكد من توافر الغاز الكافي. إذا لم يكن هناك غاز كافٍ، ستفشل العملية.
- **تكرار إنشاء العقود**: كن حذرًا عند إنشاء عقود جديدة في حلقة أو دالة قد تُستدعى عدة مرات، حيث يمكن أن يؤدي ذلك إلى استهلاك كبير للغاز وقد يؤثر على أداء الشبكة.

## ملخص في جملة واحدة
تُستخدم الكلمة المفتاحية "new" في Solidity لإنشاء مثيلات جديدة من العقود الذكية أو الهياكل، مما يعزز من قدرة التفاعل بين العقود ويساعد في بناء تطبيقات أكثر تعقيدًا.
